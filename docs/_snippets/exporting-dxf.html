--- title: DXF ---

Simple export

Call makerjs.exporter.toDXF(model) passing your model. This function returns a string of DXF.

Advanced options

Call makerjs.exporter.toDXF(model, options) passing your model and an options object. The options object has these properties:

  • units - Maker.js unit type (default - extracted from drawing. If unit system is not in drawing or not passed, it will use DXF default of inches)
  • fontSize - (default - 9) font size of captions.
